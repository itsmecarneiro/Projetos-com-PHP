# Avaliação de Produto em PHP

Mini projeto backend em PHP focado em validação de dados e regras de negócio.

## Funcionalidades
- Validação de comentário (5 a 300 caracteres)
- Validação de nota (1 a 5)
- Classificação da avaliação
- Validação de email com filter_var

## Tecnologias
- PHP 8+
- Funções nativas
- Boas práticas de backend

## Objetivo
Projeto criado para praticar lógica backend e compor portfólio para vagas júnior.
